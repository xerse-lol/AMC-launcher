AMC Launcher

Documentation and purpose for the AMC Launcher project.

AMC Launcher is a lightweight Minecraft launcher built for low end PCs. The goal is to keep RAM and CPU usage low while keeping the UI simple, fast, and clear to use.
Status: Private alpha

Purpose

AMC Launcher is designed for players on budget hardware who still want a clean, modern launcher. The priority is stable performance, small downloads, and minimal background usage.

Features

Version control

  • Install and launch official Minecraft versions.
  • Automatic library and asset downloads.
  • Clear launch logs for troubleshooting.

Accounts

  • Microsoft sign-in for full online access.
  • Local profiles for offline testing.
  • Simple account switcher.

Performance presets

  • Low-end mode tuned for 4-8 GB RAM systems.
  • Potato mode for extreme low spec devices.
  • Memory sliders with safe defaults.

Content tools

  • Mods and resource pack management.
  • Skin library for local profiles.
  • Quick folder shortcuts.

System requirements

Minimum

  • Windows 10 or 11
  • 4 GB RAM
  • Integrated graphics
  • Java 17+ (21 recommended for newer versions)

Recommended

  • 8 GB RAM
  • SSD storage
  • Java 21
  • Updated GPU drivers

Quick start

  1. Launch AMC Launcher.
  2. Select a Minecraft version.
  3. Pick your RAM range and presets.
  4. Click Launch.

Accounts

Microsoft login is required for online play. Local profiles are intended for offline testing and singleplayer only.

New third-party launchers must be reviewed by Mojang to access the Java services APIs. If Microsoft login fails with "Invalid app registration", submit the launcher client ID to the official review form: https://aka.ms/mce-reviewappid

Performance tuning

  • Use Low-end preset for smoother FPS on 4-8 GB systems.
  • Use Potato mode for maximum stability on very low specs.
  • Keep background apps closed to free RAM.

Mods and resource packs

  • Enable or disable mods directly in the launcher UI.
  • Use the quick folders to drop in new mods or packs.
  • Large modpacks may need higher RAM limits.

Privacy and security

  • Login runs on Microsoft-hosted pages. No passwords are collected by AMC.
  • Tokens are stored locally and encrypted.
  • No telemetry is shared outside the launcher.

Roadmap

  • Better mod metadata and icons.
  • Expanded skin management for Microsoft accounts.
  • More graphics presets and auto tuning.
